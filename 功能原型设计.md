# 功能原型设计（用于前端页面与交互规划）

> 目标：把「功能概览」细化成可直接用于原型图 / 界面设计的说明，覆盖典型页面、核心交互和关键字段。  
> 角色默认是「学生用户」，后续如有教师 / 管理员角色，可以在此基础上补充差异。

---

## 一、账号与认证

### 1. 登录 / 注册流程

**页面：登录页**
- 入口：未登录访问任何需要权限的页面时重定向到此页。
- 元素：
  - 顶部：App 名称 / Logo。
  - 登录方式切换 Tab：
    - 「手机号登录」
    - 「账号密码登录」
  - 手机号登录区域：
    - 输入框：手机号
    - 输入框：短信验证码
    - 按钮：「获取验证码」（带倒计时、频率限制提示）
    - 按钮：「登录」
  - 账号密码登录区域：
    - 输入框：账号（或手机号 / 邮箱）
    - 输入框：密码（带「显示/隐藏」切换）
    - 勾选：「记住我」
    - 链接：「忘记密码」
    - 按钮：「登录」
  - 第三方登录区域：
    - 按钮：「QQ 登录」「微信登录」「支付宝登录」
  - 底部文案：「还没有账号？去注册」

**页面：注册页**
- 元素：
  - 输入框：昵称
  - 输入框：手机号
  - 输入框：短信验证码（按钮「获取验证码」）
  - 输入框：设置密码
  - 输入框：确认密码
  - 勾选：同意《用户协议》《隐私政策》
  - 按钮：「注册并登录」

**页面：忘记密码页**
- 元素：
  - 输入框：手机号 / 注册账号
  - 输入框：短信验证码
  - 输入框：新密码
  - 输入框：确认新密码
  - 按钮：「重置密码」

### 2. 个人资料编辑

**页面：个人资料页（在“我的”模块中）**
- 元素：
  - 头像：可点击更换（上传图片 / 裁剪）
  - 昵称：输入框
  - 个性签名 / 自我介绍：多行输入
  - 性别、年级、专业等（可选）
  - 按钮：「保存修改」

## 二、课程表模块

### 1. 课程表主页面

**页面：课程表页**
- 顶部区域：
  - 下拉：学年学期选择（如：2024-2025 第一学期）
  - 切换控件：视图类型（「月」「周」「日」）
  - 切换控件：周次选择（第 N 周，下拉或左右切换）
  - 开关：显示计划叠加（勾选时，在课程表上叠加显示计划）
  - 按钮：「导入课程表」「导出课程表」「新增课程」
- 主体区域（以周视图为例）：
  - 顶部列：星期一至星期日（支持自定义名称）
  - 左侧列：时间轴 / 节次（如第 1 节、第 2 节，或具体时间段）
  - 网格：一个格子代表「某天某时间段」，可放多个课程 / 计划：
    - 课程卡片元素：
      - 课程名称
      - 教室 / 地点
      - 时间范围（如 8:00-9:35）
      - 教师姓名（可选）
      - 颜色块（对应课程颜色）
    - 叠加的计划标记：
      - 简短标签，如「复习」「做作业」，显示在课程卡片下方或角标。

**交互：**
- 点击课程卡片：
  - 打开「课程详情侧边栏」或弹窗。
- 鼠标悬停课程卡片：
  - 显示悬浮卡片，包含详细信息（周次范围、单双周、备注等）。
- 拖拽课程卡片：
  - 支持调整上课时间或更改日期（可在后续版本启用）。

### 2. 课程详情与编辑

**弹窗 / 侧边栏：课程详情**
- 基本信息展示：
  - 课程名称（可编辑）
  - 教师
  - 上课地点
  - 上课时间（开始时间、结束时间）
  - 周次设置（开始周、结束周、单双周）
  - 课程颜色选择器
  - 备注
- 操作按钮：
  - 「编辑」/「保存」
  - 「删除课程」
  - 「复制为新课程」（可选）
- 扩展区域：
  - Tab1：「课程资料」：文件列表（名称、类型、上传时间、下载 / 打开按钮）
  - Tab2：「课程笔记」：按日期罗列笔记，可新建/编辑/删除
  - Tab3：「课程录音」：录音列表，可播放/删除
  - Tab4：「AI 聊天」：与课程相关的聊天窗口（上下文包含课程信息）

### 3. 课程导入 / 导出

**弹窗：导入课程表**
- 元素：
  - 上传区域：支持拖拽或点击上传（Excel / CSV / 模板文件）
  - 模板下载链接：「下载课程表模板」
  - 解析预览：上传后显示解析出的课程列表（课程名、时间、周次等）
  - 下拉：选择导入到哪个学期
  - 按钮：「确认导入」「取消」

**导出功能：**
- 在课程表页点击「导出课程表」：
  - 弹出选择框：导出格式（Excel / CSV / 图片）
  - 选项：导出当前周 / 整个学期

## 三、计划模块

### 1. 计划列表与总览

**页面：计划列表页**
- 顶部筛选区域：
  - 搜索框：按计划名称关键字搜索
  - 下拉：计划类型（学习 / 项目 / 生活 / 自定义）
  - 下拉：状态（全部 / 未开始 / 进行中 / 已完成 / 已取消）
  - 日期筛选：开始日期 / 结束日期
  - 按钮：「新建计划」
- 列表区域：
  - 每条计划卡片显示：
    - 标题
    - 类型、状态标签
    - 时间范围（开始日期 - 结束日期）
    - 当天进度 / 总进度（可以简单显示“已完成 X%”）
    - 最近一次更新时间
  - 点击计划卡片：
    - 进入「计划详情页」

### 2. 计划详情与任务结构

**页面：计划详情页**
- 顶部信息：
  - 计划标题（可编辑）
  - 类型、状态（下拉切换）
  - 时间范围（开始 / 结束日期）
  - 学习目标、主要目标、预期结果摘要
- 中部 Tab：
  - Tab1：「任务列表」
  - Tab2：「日历视图」
  - Tab3：「日记 & 评价」
  - Tab4：「AI 助手」

**Tab1：任务列表**
- 列表展示：
  - 任务名称
  - 开始时间、结束时间或预计天数
  - 优先级（高 / 中 / 低）
  - 状态（未开始 / 进行中 / 已完成 / 已取消）
- 操作：
  - 新建任务：弹出任务编辑弹窗
  - 编辑 / 删除任务
  - 勾选「标记为完成」

**任务编辑弹窗：**
- 字段：
  - 任务名称（必填）
  - 描述（可选）
  - 开始时间、结束时间 / 预计用时
  - 优先级
  - 状态
  - 关联标签（多选）

### 3. 计划设定与规则

**区域：计划设定（在详情页侧边栏或单独区块）**
- 字段：
  - 整体开始时间、结束时间
  - 每日学习时长（例如：1.5 小时 / 天）
  - 提醒时间（例如：每天 19:00 提醒）
  - 学习目标（一句话版本）
  - 主要目标与预期结果（多行说明）

### 4. 日历、日记与评价

**Tab2：日历视图**
- 显示当前计划在日历上的分布：
  - 每天标注对应的任务或学习时间块
  - 点击某一天：右侧展示当天任务列表和日记入口

**Tab3：日记 & 评价**
- 当天日记：
  - 日期选择（默认今天）
  - 文本编辑区：记录当天学习过程、感受
  - 上传附件：文件 / 图片
  - 心情选择：如表情或心情标签
- 自我评价：
  - 打分组件（1–5 星或 0–100 分）
  - 「今天的收获 / 问题」文本框
  - 按钮：「保存日记」

### 5. 标签与 AI 助手

**标签管理页（或弹窗）**
- 列表：
  - 标签名称、颜色
  - 使用计数（被多少计划 / 日记使用）
- 操作：
  - 新增标签（名称 + 颜色）
  - 修改标签
  - 删除标签（考虑是否允许有绑定时删除）

**Tab4：AI 助手**
- 聊天窗口：
  - 左侧为消息列表（用户 / AI）
  - 底部输入框 + 发送按钮
  - 上传文件按钮（作为上下文）
- 上下文说明：
  - 默认附带当前计划的基本信息（标题、目标、时间范围、任务列表概览）
  - AI 回答风格遵循课程助手的 CSFE 思考框架（但对用户保持自然聊天体验）

## 四、个人中心（“我的”模块）

### 1. 个人资料与 AI 设置

**页面：我的主页**
- 上方：个人卡片
  - 头像
  - 昵称
  - 简介
  - 快捷统计（今日任务数、今日学习时长等）
- 入口卡片：
  - 「个人资料」
  - 「AI 设置」
  - 「学习统计」
  - 「我的笔记」
  - 「系统设置」

**页面：AI 设置**
- 字段：
  - 当前模型：下拉选择（如 gpt-4o / 其他）
  - 温度（滑条）
  - 最大回复长度
  - 默认提示词模板（可以选择预设 / 自定义）
  - 启用的工具 / 插件列表（勾选）
- 按钮：
  - 「恢复默认设置」
  - 「保存设置」

### 2. 学习统计与提醒

**页面：学习统计**
- 时间范围选择：本周 / 本月 / 本学期 / 自定义
- 指标卡片：
  - 累计学习时长
  - 完成计划数量
  - 课程出勤率
  - 最近 7 天学习折线图
- 列表：
  - 近期完成的计划 / 任务列表

**提醒设置（可放在系统设置或单独页面）**
- 开关：
  - 课程提醒：开启 / 关闭
  - 计划提醒：开启 / 关闭
- 具体设置：
  - 提前提醒时间：如上课前 10/20/30 分钟
  - 通知渠道：站内消息 / 邮件 / 推送（视实现能力而定）

### 3. 笔记与系统设置

**页面：我的笔记**
- 筛选：
  - 来源：课程笔记 / 计划笔记
  - 关联课程 / 关联计划
  - 时间范围
  - 标签
- 列表：
  - 笔记标题 / 前几行内容
  - 关联对象（哪门课 / 哪个计划）
  - 创建时间、最近编辑时间
- 操作：
  - 查看详情、编辑、删除

**页面：系统设置**
- 语言选择（中文 / 其他）
- 主题风格（浅色 / 深色 / 跟随系统）
- 通知开关（与提醒设置联动或统一管理）
- 数据与隐私相关入口（如导出数据、注销账号等，可根据需求扩展）
